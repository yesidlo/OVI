HTML

Lengaje tilizado para el desarrollo de páginas de Internet. Se habla, de Lenguaje de Marcas de Hipertexto, es interpretado como Lenguaje de Formato de Documentos para Hipertexto. Estándar de Lenguaje de Marcado universal que se comprende como método la posibilidad de organizar y marcar distintos documentos dentro de una lista. EL HTML se compromete entender una explicación con la capacidad que manifiesta como textos y encima de su organización, añadiendo distintos texto con diferentes cosas. El verbo es totalmente sencillo y común que ayuda para determinar otros lenguajes en lo cual tienen que ver con el formato de los documentos. El escrito en el que se inventa desde las protocolo, que acceden interconectar variado definiciones y formas. Para la redacción de este lenguaje, se inventa protocolo que se manifiesta determinada a inclinación de broche.

HEAD

Para que la documentación pueda ser visualizado en la internet, contienen las marcas (Netscape, MsIe o Explorer). El HEAD contiene etiquetas muy importante como TITLE y META.
TITLE: Sin una duda este es el el elemento mas importante de una página HTML, pues esta indica el titulo del documento y la cual es obligatoria en un archivo HTML.
En cuanto el META añade información sobre la página, como la descripción, autor, etc.

BODY

Body significa cuerpo. El Body junto al Head son los encargados de mostrar el contenido de una página web.
El body siempre va después del Head, además de esto el elemento body posee muchos atributos; aunque muchos de ellos ya han sido desaprobados con el transcurrir del tiempo con la implementación de nuevas tecnologias en favor de las hojas de estilos.

HTML5

“HTML5 es la quinta Versión del importante lenguaje HTML”. Tuvo sus inicio en 2004. Esta versión incorporó nuevos elementos, atributos y etiquetas. Html5 ha revolucionado la Web con una tecnologia mas versatil, la cual ha permitido que el desarrollar páginas Web hoy en dia se mas dinamico. Esta versión permite el perfecionamiento de las páginas web y or ende el mejor profesionalismo en los desarrolladores

Etiquetas atributos

ETIQUETAS: son la estructura básica de HTML, en ella se encuentran las etiquetas y elementos que abarca una página Web, pero a veces no abarcan la información necesaria y necesita complementarse con los atributos.

ATRIBUTOS:Los atributos ayudan en la estructuración de un página Web y indica como debe presentarse tal sitio. La mejor forma de usar los atributos en una página Web es a través de las hojas de estilo en casacada (CSS)
